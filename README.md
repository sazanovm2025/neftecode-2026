# neftecode-2026

Решение задачи хакатона «Нефтекод 2026» — прогнозирование свойств смазочных материалов по составу смеси.

## Структура репозитория

```
neftecode-2026/
├── data/              # данные (НЕ в git, см. .gitignore)
│   └── raw/           # исходные входные файлы
├── notebooks/         # исследовательские Jupyter-ноутбуки
├── src/               # переиспользуемый код
└── README.md
```

## Данные

Входные данные лежат локально в `data/raw/` и не коммитятся.
